# Go语言设计模式——选项模式

先定义`Human`结构体

```
type Human struct {
	Name string // 姓名
	Age  int    // 年龄
	Sex  int    // 性别 1=男 0=女
}
```

再定义`OptionFunc`函数类型

```
type OptionFunc func(*Human)
```

然后利用闭包为每个字段编写一个设置值的Set函数

```
func Name(name string) OptionFunc {
	return func(human *Human) {
		human.Name = name
	}
}

func Age(age int) OptionFunc {
	return func(human *Human) {
		human.Age = age
	}
}

func Sex(sex int) OptionFunc {
	return func(human *Human) {
		human.Sex = sex
	}
}
```

编写一个默认值

```
var defaultHuman = &Human{
	Name: "Eve",
	Age:  1,
	Sex:  0,
}
```

最后编写构造函数

```
func NewHuman(option ...OptionFunc) *Human {
	for _, o := range option {
		o(defaultHuman)
	}
	return defaultHuman
}
```

## 完整代码

```go
package main

import "fmt"

type OptionFunc func(*Human)

type Human struct {
	Name string // 姓名
	Age  int    // 年龄
	Sex  int    // 性别 1=男 0=女
}

func Name(name string) OptionFunc {
	return func(human *Human) {
		human.Name = name
	}
}

func Age(age int) OptionFunc {
	return func(human *Human) {
		human.Age = age
	}
}

func Sex(sex int) OptionFunc {
	return func(human *Human) {
		human.Sex = sex
	}
}

func NewHuman(option ...OptionFunc) *Human {
	var defaultHuman = &Human{
		Name: "Eve",
		Age:  1,
		Sex:  0,
	}
	for _, o := range option {
		o(defaultHuman)
	}
	return defaultHuman
}

func main() {
	h1 := NewHuman()
	fmt.Println(h1) // 输出：&{Eve 1 0}
	h2 := NewHuman(Name("Adam"), Age(18), Sex(1))
	fmt.Println(h2) // 输出：&{Adam 18 1}
}
```
